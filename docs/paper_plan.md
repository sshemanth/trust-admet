# Paper Plan

## Working Title
TRUST-ADMET: A Reproducible Framework for Trustworthy Molecular AI in ADMET Prediction

## Core Claim
Molecular AI models for ADMET prediction should be evaluated not only by predictive accuracy, but also by calibration, uncertainty, scaffold robustness, applicability domain, and explainability.
